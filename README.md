# Cryptocurrency Market Tracker

A web application for tracking real-time cryptocurrency market data using the CoinGecko API.

## Features

- Real-time cryptocurrency prices
- Historical price charts
- Search functionality
- Pagination
- Multiple timeframes
- Responsive design

## Technologies

- React
- JavaScript
- HTML
- CSS
- CoinGecko API
